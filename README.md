## Setup

1. Clone repo
2. Create `.env.local` file in the root directory of this project
3. Insert `MONGO_URI='YOUR MONGODB CONNECTION STRING'` into the `.env.local` file
4. `npm i`
5. `npm run dev`
6. Open browser at `http://localhost:3000`
